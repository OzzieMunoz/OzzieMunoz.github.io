A Beginner's Guide to GitHub


Ozzie Munoz · September 7, 2025

In this guide you will learn to:

  • Create a repository on GitHub
  • Edit files and make commits
  • Use branches and project boards to organize work
  • Open pull requests and track tasks with issues

What is GitHub?

GitHub is a website where you keep coding projects (repositories), save change history (commits), and can work with teams using branches, pull requests, issues, and project boards.

Why it matters

  • Every commit is a snapshot you can roll back to.
  • Public repos + clean READMEs show real work you’ve done.
  • Employers can see how you collaborate (issues/PRs).
  • Use GitHub Pages to put a simple site or demo online.
